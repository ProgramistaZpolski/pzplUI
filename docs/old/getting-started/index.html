Adding pzplUI to your website

Let's say this is our current website:

You add pzplUI to your website just linke any other stylesheet.
Simply put this code in your head element:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/programistazpolski/pzplui@master/style.min.css">

You can see that the font changed. That's because pzplUI contains a Universal Font Stack that makes the fonts on your website work on >99.5% computers.
