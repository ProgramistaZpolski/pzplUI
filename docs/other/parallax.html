Example Parallax

Move your mouse over the image to see the effect!

Important Notice: You will need the parallax JS Plugin to use this feature. 




<div id="parallaxImg"></div>
<style>
    #parallaxImg {
        width: 640px;
        height: 426px;
        background-image: url(./aurora.jpg);
        background-size: 140%;
    }
</style>
<script>
    parallax("#parallaxImg", 0.02);
</script>
